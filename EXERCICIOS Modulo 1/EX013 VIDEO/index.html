Videos Hospedados no Proprio Servidor

Videos Hospedados no proprio servidor podem sair caro para o cliente, por conta que cada pessoa que acessa o site usa por exemplo 9mb para baixar o video e o site tem uma frequencia de 1000 pessoas, isto da por volta de 8.5 G.

  • Tag para adicionar o Video: video
  • Para ajustar o tamanho do video tag: width
  • Para aparecer o controle tag: controls
  • Para fazer uma sequencia, tipo o navegador não é compativel com mp4 e reproduzir outro tipo, fazemos o seguite
    • usamos a tag "video" como base
    • depois colocamos dentro da tag os source src
    • na tagde source src coloamos em ordem de reproducao, e colomos o tipo de arquivo video/mkv ou video/mp4 ou video/webm
  • para colocar uma capa no video colocamos a tag "poster"

Videos Externos

videos externos são videos do youtube, Vimeo e outros,

  • a unica coisa que fazer é ir no video e clicar em compartilha, em compartilhar ten una opcao incorporar e copiar o codigo
  • Caso comece a dar erro, tente no vscode, clicar com o botao direito de index.html e abrir com live server
